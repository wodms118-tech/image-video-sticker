# 투컷

사진·영상을 위아래 반반 넣고 스티커로 꾸며서 PNG/MP4로 저장하는 무료 웹앱.
서버 없음 · 업로드 없음 · 전부 사용자 폰 안에서 처리됨.

## 배포 방법 (GitHub Pages — 영어동화책 앱과 동일)

1. GitHub에서 새 저장소 생성 (예: `twocut`, Public)
2. 이 폴더의 파일 전부 업로드 — **폴더 구조 그대로!**
   ```
   index.html
   lib/mp4-muxer.js
   stickers/manifest.json
   stickers/basic/*.png
   ```
3. 저장소 Settings → Pages → Branch: `main`, 폴더: `/ (root)` → Save
4. 1~2분 뒤 `https://아이디.github.io/twocut/` 접속 완료

## 스티커팩 추가하는 법 (마케팅 무기 💪)

1. 투명 배경 PNG 스티커들을 새 폴더에 넣기 (예: `stickers/spring/`)
   - 권장: 512×512px, 파일당 100KB 이하
2. `stickers/manifest.json`에 팩 하나 추가:
   ```json
   {
     "packs": [
       { "name": "베이직", "items": [ ...기존 그대로... ] },
       {
         "name": "봄맞이 팩 🌸",
         "items": [
           "stickers/spring/cherry-blossom.png",
           "stickers/spring/butterfly.png"
         ]
       }
     ]
   }
   ```
3. GitHub에 올리면 끝 — 모든 사용자에게 즉시 반영. "이번 주 신상 스티커팩" 릴스 소재로 재활용.

## 기능 정리

| 기능 | 상태 |
|---|---|
| 비율 | 9:16 / 3:4 / 1:1 / 4:3 / 16:9 |
| 위아래 반반 | 사진·영상 자유 조합, 드래그 이동·핀치 확대, 더블탭 리셋 |
| 스티커 | 기본팩(manifest) + 내 스티커, 드래그·회전·크기조절 |
| 누끼 스티커 | 사진 → AI 배경제거(폰 안 처리, 전송 없음) → 자동 저장 |
| 내 스티커 | 브라우저 저장(IndexedDB) + 백업 파일 내보내기/불러오기 |
| 영상 | 트림(시작~끝 드래그), 반복/마지막 정지, 전체길이 자동·5·7·15·30초 |
| 저장 | PNG / 무음 MP4(H.264, 1080p, 30fps) 기기 직접 다운로드 |

## 알아둘 것

- **MP4는 무음으로 저장됨** — 인스타 업로드 시 음악 입히는 전제. 앱 안에 안내문구 있음.
- **MP4 저장 지원 기기**: 아이폰 iOS 16.4+, 안드로이드 크롬. 미지원 기기엔 자동 안내 표시.
- **누끼 첫 사용**: AI 모델(약 40MB) 1회 다운로드로 오래 걸릴 수 있음. 이후엔 캐시돼서 빠름.
- **인앱 브라우저**(인스타/카톡 안에서 링크 열기): 파일 저장이 막힐 수 있어 상단에 "외부 브라우저로 열기" 배너 자동 표시됨.
- **내 스티커는 그 기기·그 브라우저에만 저장** — 앱 안에 안내문구 + 백업 버튼 있음.
- 이름/색 바꾸기: `index.html` 상단 `:root` 색상 변수, `<span class="logo">` 로고 텍스트 수정.
